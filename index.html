DOM Methodology

how to decide which methods to use

Test it out!

.getElementById()

The listener for this button uses .getElementById("specific-button") to identify a specific box and change its color.

.querySelector()

The listener for this button uses .querySelector("h4") to identify the h4 elements in the entire document and change their font family.

The listener for this button uses .querySelector([id="center-box"]) to identify a specific box and change its color.

The listener for this button uses .querySelector([class="row0"]) to identify any boxes with the class "row0" and change their color.

.getElementsbyClass

The listener for this button uses .getElementsbyClass("column3") to identify any boxes with the class "column3" and puts them in an array. The array is then looped through and each element's color is changed.
