DOM Methods

How different methods can affect elements

.getElementById()

The listener for this button uses .getElementById("specific-box") to identify a specific box and change its color.

.getElementsByClassName()

The listener for this button uses .getElementsByClassName("column3") to identify any boxes with the class "column3" and puts them in an array. The array is then looped through and each element's color is changed.

The listener for this button uses .getElementsByClassName("box") to identify all the boxes and then puts them in an array. The array is then looped through and each element's color is changed.

.querySelector()

The listener for this button uses .querySelector("h3") to identify the first h3 element it finds in the document and change its font family and color.

The listener for this button uses .querySelectorAll("h3") to identify all the h3 elements in the entire document and change their font family and color.

The listener for this button uses .querySelector("#center-box") to identify a specific box and change its color. This can also be done with .querySelector([id="center-box"])

The listener for this button uses .querySelectorAll(".row0") to identify any boxes with the class "row0" and change their color. It can also be done with .querySelectorAll([class="row0"])
